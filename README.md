## 1. 프로젝트 주제

### 무신사 기반 패션 플랫폼

사용자의 지역에 따라서 옷을 추천해주고, 자신의 스타일 정보를 공유하는 패션 플랫폼

## 2. 기술 스택

- 백엔드
    - Python 3.10
    - Django 4.1.4
    - Django Rest Framework 3.14
    - Django Rest Framework simple-jwt 5.2.2
    - Docker 20.10.12
    - Nginx 1.22.0
    - Gunicorn 20.1.0
- 프론트엔드
    - HTML5
    - Javascript
    - JQuery
    - CSS
- 데이터베이스
    - AWS RDS PostgreSQL
    - AWS S3
- 배포
    - AWS EC2
    - AWS Route 53
    - AWS CloudFront
